# OpenGL Template

A minimal template repo for starting new OpenGL projects. Uses **GLFW** for windowing and **GLAD** for OpenGL loading.

## Stack

- **GLFW** — window and input
- **GLAD** — OpenGL function loader

## Starting a New Project

1. Clone or copy this repo.
2. Edit `src/main.cpp` (and add new sources as needed).
3. Update the build task in `.vscode/tasks.json` if you add source files.
4. Keep `glfw3.dll` in the same directory as your executable when running.
